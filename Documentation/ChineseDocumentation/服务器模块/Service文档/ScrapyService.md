# ScrapyService
---

### 执行Scrapy脚本
**方法名称**： `ExecuteScrapy`

**方法描述**：
1. 传入参数调用Python脚本后台爬取数据（关于内容实现查看Scrapy模块）
2. 返回布尔值

**请求参数**： 
| 参数名      | 类型     | 必填 | 描述         |
| ----------- | -------- | ---- | ------------ |
| searchTerms | string[] | 是   | 中英文商品名 |
| spiders     | string[] | 是   | 使用的爬虫 |

**返回数据**
 | 类型    | 描述     |
 | ------- | -------- |
 | Boolean | 爬取状态 |
